# Rekrutt - Talent Acquisition System

Rekrutt is a robust system designed to streamline the entire recruitment lifecycle, from initial hiring need to final candidate evaluation. It offers comprehensive features for detailed candidate profiling, efficient proposal handling, and end-to-end recruitment management, including scheduling, application tracking, and interviewer evaluations.

## Features

Manage users

- Users are authenticated with long-live token-based authentication
- Create users (password generated by system).
- Suspend/retire/reactiavate users.

Manage candidate profiles, work histories and related documents

- Create candidates and upload related documents.
- Blacklist/reactivate candidates.

Manage proposals and related documents

- Create proposals for hiring need.
- Submit proposals for approval.
- Reveiw submitted proposals by approve or reject.

Manage recruitments and recruitment's applications

- Create recruitments from approved proposals.
- Schedule recruitments with specific open/close timeline.
- Manually push for early open/close.
- Attach candidates to opening recruitments.
- Update candidate's application statuses (interviews, offers, discarded/withdrawn)

Manage interviews, interview's participants and interview's evaluations

- Create interviews with participants (interviewers).
- Schedule/cancel interviews.
- Evaluate succeeded interviews (by participants).
